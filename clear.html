clear(清除)屬性是用來抵銷float屬性的作用。可能的值為:

  • left:消除左邊的浮動。
  • right:消除右邊的浮動。
  • both:消除左邊及右邊的浮動。
  • none:不消除任何一邊的浮動。


來看一下從float那一頁稍作修改的例子:

CSS宣告

#leftflo at {
  float:left;
}

#clearleft {
  clear:left;
}
